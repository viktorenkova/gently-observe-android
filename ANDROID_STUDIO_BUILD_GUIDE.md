# Пошаговая инструкция: Сборка APK в Android Studio

## Важно понимать

**Build → Build APK(s)** применяется ко **ВСЕМУ ПРОЕКТУ**, а не к отдельному файлу!

Android Studio собирает все файлы проекта (Java-код, XML-layouts, ресурсы) в один APK файл.

---

## Пошаговая инструкция

### Шаг 1: Открыть проект

1. Запустите **Android Studio**
2. Выберите **Open** (не New Project!)
3. Найдите папку проекта: `OneDrive/Рабочий стол/m.app`
4. Нажмите **OK**

**Важно:** Открывайте именно папку `m.app`, а не какой-то файл внутри неё. Это папка с целым проектом.

---

### Шаг 2: Дождаться синхронизации

После открытия Android Studio:
1. Начнётся загрузка Gradle (внизу окна показывается прогресс)
2. Появятся окна синхронизации проекта
3. Дождитесь надписи **"Gradle sync finished"** внизу справа

**Это может занять 5-15 минут при первом запуске!**

---

### Шаг 3: Сборка APK

После успешной синхронизации:

#### Способ A: Меню Build

1. В верхнем меню нажмите **Build**
2. В выпадающем меню наведите на **Build Bundle(s) / APK(s)**
3. Выберите **Build APK(s)**
4. Нажмите

```
Build
  └── Build Bundle(s) / APK(s)
        └── Build APK(s)  ← нажать сюда
```

#### Способ B: Горячая клавиша

Нажмите: **Ctrl + F9** (Windows/Linux) или **Cmd + F9** (Mac)

---

### Шаг 4: Найти готовый APK

После успешной сборки появится уведомление:
> "Build Analyzer detected..." или "Build completed successfully"

Нажмите на ссылку в уведомлении или найдите файл:

```
m.app/
└── app/
    └── build/
        └── outputs/
            └── apk/
                └── debug/
                    └── GentleObserver-v1.0-debug.apk  ← ваш APK!
```

---

## Если у вас несколько модулей (реже)

Если Android Studio показывает несколько модулей (app, framework, и т.д.):

1. Слева вверху есть выпадающий список (обычно показывает "app")
2. Убедитесь, что выбран модуль **"app"**
3. Тогда Build → Build APK соберёт только приложение

---

## Возможные ошибки

### "Gradle sync failed"
- Нажмите **File** → **Sync Project with Gradle Files**
- Или перезапустите Android Studio

### "Build failed"
- Проверьте подключение к интернету (Gradle скачивает зависимости)
- Проверьте, что Java JDK установлен корректно

### "Could not find gradle-wrapper.jar"
- Откройте в VS Code → запустите `setup-and-build.ps1`
- Или скачайте Gradle вручную

---

## Альтернатива: Командная строка

Если Android Studio не работает, можно собрать через терминал:

```bash
# В папке проекта (m.app)
./gradlew assembleDebug

# Или на Windows:
gradlew.bat assembleDebug
```

APK будет создан там же: `app/build/outputs/apk/debug/`

---

## Итог

| Что вы делаете | К чему применяется |
|----------------|------------------|
| Build → Build APK(s) | Ко всему проекту (всем файлам сразу) |
| Edit → ... | К конкретному файлу |
| Run → Run 'app' | Собирает проект и устанавливает APK на устройство |

**Важно:** Вам НЕ нужно выбирать конкретный файл для сборки. Сборка — это процесс компиляции ВСЕГО проекта воедино.
